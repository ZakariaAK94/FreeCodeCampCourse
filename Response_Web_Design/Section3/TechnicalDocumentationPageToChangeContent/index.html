Introduction to HTML

HTML, which stands for HyperText Markup Language, is the foundational language for creating and structuring content on the web. It allows you to organize text, images, links, and other multimedia elements within a web browser. The term HyperText refers to the ability to create links between different documents, while Markup means that the language annotates the content, describing its structure and presentation. HTML is not a programming language; rather, it's a markup language designed to display information in a web browser in a readable and well-organized format.

Basic Structure of an HTML Document

The structure of an HTML document is essential for creating a well-functioning webpage. Every HTML document starts with the <!DOCTYPE html> declaration, which informs the browser about the version of HTML that the page is using. This is followed by the root <html> element, which encompasses the entire content of the document. Within the <html> element, there are two main sections:

  • The<head>section: This contains metadata about the document, such as its title, character set, and links to external stylesheets or scripts. Metadata does not appear on the webpage but provides essential information to browsers and search engines.
  • The<body>section: This contains all the content that users will see on the webpage, such as text, images, and interactive elements like forms and buttons.

Here's a basic example of the structure:

      
        <!DOCTYPE html>
               <html>
                   <head>
                       <title>programiz</title>
                   </head>
                   <body>
                       <h1>HTML Tutorial</h1>
                       <p>You'll learn about HTML.</p>
                   </body>
               </html>
HTML Elements and Tags

HTML uses a system of elements, also called tags, to define the structure and content of a webpage. Every element is written between angle brackets (< and >) and usually comes in pairs: an opening tag (<p>) and a closing tag (</p>). The content of the element sits between the opening and closing tags. However, some elements, such as <img> and <br>, are self-closing and do not require a closing tag. Elements can have attributes that provide additional information. For example, an <img> tag can include a src attribute to define the image source, and an alt attribute to provide alternative text for accessibility: 

          
              <img src="image.jpg" alt="Description of the image">
Common HTML Tags

HTML offers a variety of tags to organize and present different types of content. Below are some of the most commonly used tags:

  • Headings:HTML provides six levels of headings (<h1> to <h6>), with <h1> being the most important and <h6> the least. They help structure the content hierarchically and improve SEO.
    •             
                <h1>This is a Main Heading</h1>
                <h2>This is a Subheading  </h2>
  • Paragraphs: The <p> tag defines a paragraph of text.
    •             
                <p>This is a paragraph of content.</p>
  • Links: The <a> tag creates hyperlinks. The href attribute defines the destination URL.
    •             
                <a href="https://www.example.com">Visit Example</a>
  • Images: The <img> tag is used to embed images. It requires the src attribute for the image URL and the alt attribute for accessibility.
    •               
                  <img src="image.jpg" alt="A description of the image">
  • Divisions and Spans: The <div> and <span> tags are generic containers used for styling and layout purposes. <div> is a block-level element, while <span> is inline. 
    •               
                   <div>This is a block-level element. </div>
                   <span>This is an inline element.    </span>
  • <h1>— heading of the document
  • <p>—paragraph of the document
HTML Forms

HTML forms are used to collect user input and submit it to a server for processing. Forms are an essential component for building interactive websites, such as those with login systems, search bars, and feedback forms. The <form> element wraps the form controls, such as text fields, checkboxes, radio buttons, and submit buttons. Key attributes of the <form> tag include:

  • action: Specifies the URL where the form data should be sent.
  • method: Determines how the form data is sent (usually GET or POST).
    •           
            <form action="/submit-form" method="POST">
            <label for="name">Name:</label>
            <input type="text" id="name" name="name">
              
            <button type="submit">Submit</button>
            </form>
  • Inputs: The <input> tag is versatile, supporting different types of user input like text, password, checkbox, and radio.
  • Labels: The <label> tag is used to associate text with form controls, improving accessibility.
HTML Semantic Elements

Semantic HTML refers to using HTML elements that have meaning beyond their appearance. These elements clearly describe their purpose to both developers and browsers, enhancing the accessibility, SEO, and maintainability of a website. For example, the <header> element defines the header of a webpage or section, while <article> is used for standalone content, and <footer> defines the footer.

Other common semantic elements include:

  • <nav>: Represents a navigation block, such as a menu.
  • <section>: Defines a thematic grouping of content, typically with a heading.
  • <aside>:Represents content that is tangentially related to the main content, such as sidebars.
          
            <article>
              <header>
               <h1>Article Title</h1>
               <p>Published on: January 1, 2024</p>
              </header>
             <p>This is the main content of the article...</p>
            </article>
HTML Media(Images-Audio-Video)

HTML supports multimedia content like images, audio, and video to make webpages more dynamic and engaging.

  • Images: The <img> tag is used to display images. You can specify the source (src), alternative text (alt), and size (width and height).
  • Audio: HTML5 introduced the <audio> tag for embedding sound files. It supports multiple formats such as MP3, WAV, and OGG. 
    •             
              <audio controls>
              <source src="audiofile.mp3" type="audio/mp3">
              </audio>
  • Video: The <video> tag allows embedding videos. You can control the playback using attributes like controls, autoplay, and loop.
    •               
                <video controls>
                <source src="videofile.mp4" type="video/mp4">
                  </video>

    These multimedia elements help in creating rich, interactive experiences for users without relying on external plugins like Flash.
